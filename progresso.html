Prévia do Projeto

Home page

A página inicial foi pensada para simplificar a navegação, especialmente para usuários com pouca familiaridade com tecnologia. O foco é aprimorar a interface, elevando a qualidade de UI/UX e garantindo um layout totalmente responsivo para oferecer uma experiência fluida em qualquer dispositivo.

Área de cadastro

Na área de cadastro de receitas, a ideia é reduzir o esforço do usuário com atalhos que permitam registrar tudo em poucos cliques e de forma intuitiva. O próximo passo é integrar um banco de dados com os ingredientes mais usados, permitindo autopreenchimento eficiente.

Home/Culinarista

A proposta é criar uma página que torne o trabalho diário mais rápido e fluido, com abas intuitivas. Os focos incluem fichas técnicas ágeis, relatórios simplificados e uma base própria de receitas do culinarista.
